Configuración del Bot

Para poder utilizar el bot, necesitas configurar el nombre y el token del bot. Si no tienes un bot, puedes crear uno en Telegram.

Una vez que tengas el bot, copia el token y pégalo en el campo correspondiente. El nombre del bot es el nombre que le diste al bot cuando lo creaste.

Una vez que hayas configurado el bot, puedes utilizarlo para enviar mensajes a través de Telegram. para recibir tus datos.
